
# Finance
### Before Starting

- Make sure you have installed npm and node.js
- To get the project up and running, execute `npm install` in terminal
- Run the project with `npm run dev`
### Tech Stack

💻**Frontend:** React + Vite, TailwindCSS v4

🛠️**Backend:** Node.js

🗄️**Data Base**: MongoDB

📚**Libraries**: daisyUI, Lucide
